# Verbifex

A word chaining game for word lovers.

# How to play

The game starts with a word.

Words must be valid English words and meet the gameâ€™s requirements (e.g., word length of 5 characters).

The game continues until player cannot provide a valid word in the given time (15 seconds).

# Features

Validates words against an English word list.

Includes a computer opponent that generates words automatically.

Aiming to support customizable word length and time allotment, as well as multiplay.
